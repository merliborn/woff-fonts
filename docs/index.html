GitHub: woff-fonts

説明

主に個人的な目的で作成された、woff/woff2形式に変換されたフォントファイル群です。

各フォントはそれぞれのライセンスに従って公開/再配布されているため、使用したいフォントのライセンスを必ず確認してください。

GPLでライセンスされたフォントはこちらを参照してください。

フォント一覧・サンプル

Ny Stormning

ラテン文字範囲にグリフが割り当てられているルーン文字フォントです。300, 400, 700, 900の4ウェイトがあります。

Ny STormNiNg

Ny STormNiNg GreN

読み込みコード

@import url('https://merliborn.github.io/woff-fonts/css/ny-stormning.css');

Stormning Aesir

ラテン文字範囲にグリフが割り当てられているルーン文字フォントです。300, 400, 700の3ウェイトがあります。

STORMNING AESIR

読み込みコード

@import url('https://merliborn.github.io/woff-fonts/css/stormning-aesir.css');

Noto Sans Roman Numerals (Noto Sans Symbols)

Noto Sans Symbolsフォントのうち、ローマ数字に対応するU+2160-2188だけを抽出したフォントファイルです。

私のスキル上の問題により、ウェイトが可変ではなく100-900の9種類のみになっています。

ⅯⅭⅯⅬⅩⅩⅢ

読み込みコード

@import url('https://merliborn.github.io/woff-fonts/css/notosans-romannum.css');
